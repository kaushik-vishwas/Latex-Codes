# LaTeX Codes — Q&A

Home page with photo cards. Click a card to open that topic’s answer code.

## Pages

- `/` — home with clickable photo cards
- `/code/:id` — question + copyable answer code

## Add questions

Edit **`src/data/questions.js`**

## Card photos / blurbs

Edit **`src/data/homeCards.js`** — images and short text for each card (matched by question `id`).

## Run

```bash
npm install
npm run dev
```
